DOM : Document Object model
    1. HTML 문서의 구조를 표현하는 객체
    2. Element : HTML 마커업을 의미
    3. Attriute : html 마크업들의 속성

React DOM : Virtual Document object model
    1. 실제 DOM을 수정하는 것이 아닌 가상 DOM을 생성하여 수정/업데이트함
    2. Element : 컴포넌트
    3. Attriute : props

[ Life Cycle ]
    1) 컴포넌트가 생성되어 사라지기까지의 전과정
    2) 주기(시점)
        (1) mount·출생
            : main.jsx가 컴포넌트를 그려주는 함수 => .render()
            : 상위 컴포넌트가 return할 때, 하위 컴포넌트가 같이 return 되므로 하위컴포넌트도 출생됨
            
            useEffect( () => {}, [] ) // 컴포넌트가 출생될 때, 딱 1번만 실행

        (2) update·인생
            : 컴포넌트가 재실행-rerendering 됨
            = 화면 새로고침 = 컴포넌트 재호출 = 함수 재호출
            useEffect( () => {}, [의존성 배열] )
                의존성 배열 : 참조 목록으로 특정한 상태가 변경될 때

        (3) unmount·사망
            : 컴포넌트가 화면에서 사라질 때, (별도로 관리할 필요가 없음)
