[ 본체 ]
    - 메모리(RAM)      : 휘발성 메모리, 주기억장치로써 실행중인 상태에서만 메모리 저장
    - SSD/HDD(ROM)    : 비휘발성 메모리, 보조기억장치이면서 비실행중인 상태에서 메모리 저장
    * 보조기억 장치는 용량잉 크고, 주 기억장치는 속도가 빠름

[ 개발 학습상 주요 저장 ]
    - 저장하고자 하는 데이터의 사용처를 6하원칙에 맞추어 생각하기
    - 1 개월차 : 크롬의 로컬스토리지 / 세션저장소     : 클라이언트 환경에 저장 - 보안에 취약
    - 2 개월차 : DB                               : 영구저장, 자바 서버 외 DB서버에 저장
    - 3 개월차 : 톰캣(Spring) 세샨                 : 자바 서버에 저장, 서버 과부하, 보안에 안전, 로그인 정보
    - 5 개월차 : 레디스                            : 임시저장(캐싱), 다중 서버 - DB 조회 결과 임시 저장, 실시간 정보
    + csv

[ Redis ]
    1) 정의
        : 인메모리 기반의 데이터 저장소
    2) 목적
        : 관계형 DB가 보조기억장치이므로 빠른 메모리 처리가 필용·사용할 때,
            MySQL/ORCLE : SQL - 관계형DB
            REDIS : NOSQL - 객체지향
    3) 구조
        : Key-Value, Map/JSON/DTO 구조와 유사
    4) 캐싱
        : select 할 때 마다 DB에 접근·요청에 따라 속도가 느리거나 성능 저하가 발생할 수 있으므로
            한 번 select 한 결과를 캐싱
    5) 설치
        (1) Spring 내장 Redis 인메모리 - 안함
        (2) Redis 설치형
            - 윈도우/로컬 : https://github.com/redis-windows/redis-windows/releases?utm_source=chatgpt.com
            - Assets 에서 버전 선택
                : Redis-8.2.2-Windows-x64-msys2-with-Service.zip : 백그라운드 지원
                : Redis-8.2.2-Windows-x64-msys2.zip             : 직접 실행 - 강의 기준
            - 압축해제 > 레디스 폴더 내에 redis-server.exe 실행
                - 끄면 종료 = 서버 off = 인메모리/휘발성이므로 저장된 모든 정보가 사라짐
                - Redis port : 6379

            - Spring <> Redis 연동 - build.gradle 주입
                `implementation 'org.springframework.boot:spring-boot-starter-data-redis'`
        (3) Application.properties 설정
            ```
                # [6] redis 서버 연동
                spring.data.redis.host=localhost
                spring.data.redis.port=6379
            ```

    6) 주 사용법
        (1) 레디스 템플릿 객체를 생성
            ` private final StringRedisTemplate stringRedisTemplate; `  : 문자열 타입
            ` private final RedisTemplate redisTemplate; `              : 그외 타입
            >> ` private final RedisTemplate<String, Object> redisTemplate; `
            ** RedisTemplate 사용할 경우, 레디스에 저장할 타입 설정 필요
                ```java
                    @Configuration
                    public class RedisConfig {
                        // RedisTemplate 관련 설정 메소드
                        @Bean
                        public RedisTemplate<String,Object> redisTemplate(RedisConnectionFactory connectionFactory){
                            // [1.1] Redis 템플릿 객체 생성
                            // Redis 형식을 Map 타입으로 사용하기 위한 설정
                            RedisTemplate<String, Object> template = new RedisTemplate<>();
                            // [1.2] template 를 저장소 factory 에 저장
                            template.setConnectionFactory( connectionFactory );
                            // 직렬화 : 레디스에 저장된 데이터를 자바 타입으로 변환하는 과정
                            // [1.3] template 의 key 값을 String 타입으로 직렬화
                            template.setKeySerializer(new StringRedisSerializer());
                            // [1.4] template 의 value 값을 JSON/DTO 타입으로 직렬화
                            template.setValueSerializer(new GenericJackson2JsonRedisSerializer());
                            return template;
                        } // func end
                    } // class end
                ```
        (2) 레디스에 데이터를 저장하는 방법
            1] redisTemplate.opsForValue().set(key, value);
            2] redisTemplate.opsForValue().set(key, value, TTL);
                * TTL·유효기간 : 지정한 기간, 시간이 지나면 자동으로 삭제
                    Duration.of단위(숫자)
                    `Duration.ofSecond(10)` 유효기간 10초
                * key는 중복 불가!

        (3) 레디스에 데이터를 호출하는 방법
            1] 특정 key의 value 반환
                ` Object value = redisTemplate.opsForValue().get(key); `
            2] 모든 key 반환
                ` Set<String> keys = redisTemplate.keys("*"); `
            3] 특정 패턴의 key 반환
                ` Set<String> keys - redisTemplate.keys("패턴"); `
                * 관례적으로 redis의 key 명칭은 `의미상단어:식별번호`
                    ` student:1, student:2, student:3, ... `
                    ` Set<String> keys - redisTemplate.keys("student:"); `

        (4) 레디스 데이터를 삭제하는 방법
            1] 특정 엔트리 삭제
                ` boolean result = redisTemplate.delete("key);"; `
        (5) 레디스 key 존재 여부 확인 방법
            ` boonlean result = redisTemplate.hasKey(Key); `
